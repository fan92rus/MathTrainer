# Roadmap типов заданий MathTrainer

> Документ описывает потенциальные типы заданий для добавления в приложение MathTrainer, приоритизированные по важности для обучения учеников 1-4 классов.

---

## 📊 Легенда приоритетов

| Приоритет | Описание |
|-----------|----------|
| 🔴 **P0 — Критический** | Базовый навык, без которого невозможен дальнейший прогресс в математике |
| 🟠 **P1 — Высокий** | Важный навык, часто встречается в учебной программе и日常生活 |
| 🟡 **P2 — Средний** | Полезный навык, расширяющий математическую грамотность |
| 🟢 **P3 — Низкий** | Дополнительный навык, углубляющий понимание математики |

---

## 🔴 P0 — Критические (базовые навыки)

### 1. Сложение в столбик

| Характеристика | Значение |
|----------------|----------|
| **Классы** | 2-4 |
| **Связь с текущим** | Вычитание в столбик уже реализовано |
| **Сложность реализации** | 🟢 Низкая — код можно переиспользовать |

#### Почему это критически важно:

1. **Зеркальная операция** — вычитание в столбик уже есть, сложение — обратная операция, логично завершить набор
2. **Фундамент для многозначных чисел** — без сложения в столбик невозможна работа с числами > 100
3. **Школьная программа** — это обязательный навык во 2 классе
4. **Повседневная необходимость** — подсчёт денег, сумм покупок

#### Уровни сложности:
- Уровень 1: Двузначные без перехода через десяток (23 + 15)
- Уровень 2: Двузначные с переходом через десяток (48 + 37)
- Уровень 3: Трёхзначные без перехода (245 + 132)
- Уровень 4: Трёхзначные с переходом (567 + 284)
- Уровень 5: Сложение нескольких чисел (123 + 456 + 78)

---

### 2. Деление (начальный уровень)

| Характеристика | Значение |
|----------------|----------|
| **Классы** | 3-4 |
| **Связь с текущим** | Таблица умножения уже реализована |
| **Сложность реализации** | 🟡 Средняя — требуется генерация с проверкой делимости |

#### Почему это критически важно:

1. **Обратная операция умножения** — таблица умножения есть, деление — естественное продолжение
2. **Завершение набора базовых операций** — после +, -, × остаётся только ÷
3. **Фундамент для дробей** — без понимания деления невозможна работа с дробями
4. **Школьная программа** — деление вводится в 3 классе, это обязательный навык

#### Уровни сложности:
- Уровень 1: Деление в пределах таблицы умножения (42 ÷ 6)
- Уровень 2: Деление с остатком (17 ÷ 5 = 3 ост. 2)
- Уровень 3: Деление двузначных на однозначное (84 ÷ 4)
- Уровень 4: Деление трёхзначных на однозначное (156 ÷ 3)
- Уровень 5: Деление с нулём в частном (408 ÷ 4)

---

### 3. Сравнение чисел и выражений

| Характеристика | Значение |
|----------------|----------|
| **Классы** | 1-4 |
| **Связь с текущим** | Универсальный навык для всех типов заданий |
| **Сложность реализации** | 🟢 Низкая — простая генерация и UI |

#### Почему это критически важно:

1. **Фундамент математического мышления** — понимание отношений "больше", "меньше", "равно"
2. **Необходимость в каждом типе заданий** — проверка ответа — это сравнение
3. **Подготовка к уравнениям** — сравнение выражений развивает алгебраическое мышление
4. **Школьная программа** — начинается с 1 класса, продолжается всё время

#### Уровни сложности:
- Уровень 1: Сравнение чисел до 10 (7 ○ 5)
- Уровень 2: Сравнение чисел до 100 (67 ○ 76)
- Уровень 3: Сравнение чисел до 1000 (456 ○ 465)
- Уровень 4: Сравнение выражений (15 + 3 ○ 20 - 2)
- Уровень 5: Сложные выражения (5 × 4 ○ 10 + 8)
- Уровень 6: Цепочки сравнений (5 < 10 < 15 < 20)

## 🟠 P1 — Высокий приоритет (важные навыки)

### 4. Сложение и вычитание величин (измерений)

| Характеристика | Значение |
|----------------|----------|
| **Классы** | 2-4 |
| **Связь с текущим** | Расширяет арифметику на реальные величины |
| **Сложность реализации** | 🟡 Средняя — разные единицы измерения |

#### Почему это важно:

1. **Практическая применимость** — необходимые навыки для повседневной жизни
2. **Связь математики с реальностью** — абстрактные числа становятся осязаемыми
3. **Школьная программа** — обязательная тема во 2-4 классах
4. **Двухэтапное мышление** — сначала перевод единиц, потом вычисление

#### Подтипы:
- **Длина**: мм, см, дм, м, км — переводы и вычисления
- **Время**: ч, мин, с, сутки — "через 15 мин", "сколько минут в 2 часах"
- **Масса**: г, кг — "5 кг 200 г + 400 г"
- **Стоимость**: руб, коп — практические задачи с деньгами

---

### 5. Разрядные слагаемые

| Характеристика | Значение |
|----------------|----------|
| **Классы** | 2-3 |
| **Связь с текущим** | Углубляет понимание "Вычисления удобным способом" |
| **Сложность реализации** | 🟢 Низкая — простая генерация |

#### Почему это важно:

1. **Основа устного счёта** — разрядное сложение = удобный способ вычислений
2. **Понимание позиционной системы** — зачем важны нули и разряды
3. **Подготовка к столбику** — понимание алгоритма сложения в столбик
4. **Школьная программа** — стандартная тема 2 класса

#### Уровни сложности:
- Уровень 1: Представь число (23 = 20 + 3)
- Уровень 2: Собери число из разрядов (400 + 50 + 7 = ?)
- Уровень 3: Сложение разрядных слагаемых (300 + 20 + 400 + 50)
- Уровень 4: Сравнение разрядных представлений (245 = 200 + 40 + 5)

### 6. Умножение в столбик

| Характеристика | Значение |
|----------------|----------|
| **Классы** | 3-4 |
| **Связь с текущим** | Таблица умножения реализована, вычитание в столбик есть |
| **Сложность реализации** | 🟡 Средняя — требуется реализация алгоритма |

#### Почему это важно:

1. **Естественное продолжение таблицы** — после освоения умножения до 10, ученики переходят к многозначным числам
2. **Завершение набора столбичных операций** — вместе со сложением и вычитанием образует базу
3. **Школьная программа** — обязательный навык с 3 класса
4. **Практическая необходимость** — подсчёт площадей, стоимости, количества
5. **Фундамент для сложных вычислений** — используется в умножении величин, дробей

#### Уровни сложности:
- Уровень 1: Двузначное на однозначное (23 × 4)
- Уровень 2: Двузначное на двузначное (24 × 13)
- Уровень 3: Трёхзначное на однозначное (156 × 5)
- Уровень 4: Трёхзначное на двузначное (234 × 15)
- Уровень 5: С нулями в середине (205 × 4, 340 × 6)
- Уровень 6: Четырёхзначное на однозначное (1823 × 7)

---

### 7. Порядок действий (смешанные операции)

| Характеристика | Значение |
|----------------|----------|
| **Классы** | 3-4 |
| **Связь с текущим** | Объединяет все арифметические операции |
| **Сложность реализации** | 🟡 Средняя — требуется генератор выражений |

#### Почему это важно:

1. **Ключевой навык для математики** — без этого невозможна работа со сложными выражениями
2. **Интеграция всех операций** — ученик применяет всё изученное вместе
3. **Школьная программа** — критически важная тема с 3 класса
4. **Логическое мышление** — учит планировать последовательность действий

#### Уровни сложности:
- Уровень 1: Сложение и умножение (5 + 3 × 4)
- Уровень 2: Вычитание и деление (20 - 12 ÷ 3)
- Уровень 3: Четыре действия без скобок (25 + 15 ÷ 5 - 3)
- Уровень 4: Со скобками (2 × (15 - 7) + 10)
- Уровень 5: Сложные выражения ((100 - 40) ÷ (12 - 8))

### 8. Простые текстовые задачи

| Характеристика | Значение |
|----------------|----------|
| **Классы** | 1-4 |
| **Связь с текущим** | Применение всех арифметических навыков |
| **Сложность реализации** | 🔴 Высокая — нужны шаблоны задач и парсинг |

#### Почему это важно:

1. **Связь математики с жизнью** — демонстрирует практическую ценность навыков
2. **Развитие мышления** — перевод текста в математическую модель
3. **Школьная программа** — основная часть всех контрольных работ
4. **Мотивация** — ученики видят смысл в абстрактных вычислениях

#### Подтипы:
- **На нахождение суммы/разности**: "У Маши 5 яблок, ей дали ещё 3. Сколько стало?"
- **На сравнение**: "У Пети 12 марок, у Васи — 8. На сколько больше у Пети?"
- **На увеличение/уменьшение**: "Было 15, взяли несколько, осталось 7. Сколько взяли?"
- **На кратное сравнение**: "Во сколько раз 12 больше 4?"
- **На цену/количество/стоимость**: "3 ручки по 15 рублей. Сколько стоят?"

## 🚀 Рекомендуемый порядок реализации

### Этап 1: Быстрый выигрыш (1-2 недели)
Цель: Закрыть критические пробелы в базовых операциях

1. **Сравнение чисел и выражений** — самый простой, даёт сразу пользу
2. **Разрядные слагаемые** — легко реализовать, полезно для устного счёта
3. **Сложение в столбик** — переиспользование кода вычитания

### Этап 2: Завершение базовых операций (2-3 недели)
Цель: Все 4 арифметических действия полностью

4. **Деление (без столбика)** — замыкает набор +, -, ×, ÷
5. **Сложение/вычитание величин** — практическое применение арифметики

### Этап 3: Комплексные навыки (3-4 недели)
Цель: Применение знаний в новых форматах

6. **Умножение в столбик** — естественное продолжение таблицы умножения
7. **Порядок действий** — интеграция всех операций
8. **Периметр и площадь** — геометрический компонент
9. **Числовые ряды** — алгоритмическое мышление

### Этап 4: Сложные форматы (4+ недель)
Цель: Продвинутые навыки

10. **Деление в столбик** — сложный, но важный навык
11. **Текстовые задачи** — новый формат, требует генератора
12. **Дроби** — следующий уровень после деления

### Этап 5: Дополнительные функции (по мере необходимости)
Цель: Расширение функционала

13-16: Римские цифры, время, игровые форматы

### Технические рекомендации

1. **Переиспользование кода**:
   - Сложение в столбик → переиспользовать компоненты вычитания
   - Умножение в столбик → переиспользовать логику сложения в столбик
   - Деление → переиспользовать логику умножения
   - Величины → общая система конвертации единиц

2. **Конфигурация доступности**:
   - Все типы заданий добавить в `exerciseAvailability.ts`
   - Настроить доступность по классам и четвертям

3. **Единый стиль**:
   - Следовать паттерну существующих `src/utils/math/*/index.ts`
   - Использовать те же типы из `src/types/index.ts`

4. **Тестирование**:
   - Каждый тип заданий требует набора тестов
   - Проверка граничных случаев (нуль, единица, максимальные числа)
